Function gt

  • gt(x: _Interval, y: _Interval): boolean

  • Checks if the interval x is greater than y i.e. if all the values of x are greater than the right endpoint of y

    Example

    Interval.greaterThan(
      Interval(2, 3),
      Interval(4, 5)
    ) // false

    Example

    Interval.greaterThan(
      Interval(4, 5),
      Interval(2, 3)
    ) // true
